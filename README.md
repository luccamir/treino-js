# 🏋️ Sistema de Montagem de Treino

Exercício prático desenvolvido como atividade da FIAP, simulando um sistema interativo de montagem de treino personalizado via terminal do navegador.

## 📋 Sobre o Projeto

O sistema coleta informações do aluno por meio de prompts e monta um treino personalizado com base nas respostas, incluindo exercícios, repetições e feedback do usuário.

## ⚙️ Funcionalidades

- Cadastro do aluno e nome do treino
- Registro de 3 exercícios com número de repetições
- Feedback do aluno sobre o treino (bom, ruim ou neutro)
- Opção de montar um segundo treino
- Validação de respostas inválidas

## 🛠️ Tecnologias

- JavaScript (Vanilla)

## 🚀 Como executar

1. Abra o Google Chrome
2. Pressione `F12` → aba **Console**
3. Cole o código e pressione **Enter**
